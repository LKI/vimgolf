# Introduction

Vim strokes for [VIM-GOLF][1]

# Quick Start

Goto [Vim-Golf][1] and sign in with twitter.

>Real Vim ninjas count *every* keystroke - do you?

>Pick a challenge, fire up Vim, and show us what you got.

Run:

```
gem install vimgolf
vimgolf setup
vimgolf put [challenge-id]
```

[1]:http://vimgolf.com
